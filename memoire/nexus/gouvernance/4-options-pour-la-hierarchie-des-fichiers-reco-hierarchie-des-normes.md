# 4 options pour la hierarchie des fichiers + reco (hierarchie des normes) — domaine: nexus / catégorie: gouvernance
> Créé le 22/06/2026 · Dernière mise à jour le 22/06/2026

## En bref
4 options pour ordonner les fichiers. RECO = C : hierarchie des normes constitutionnelle (identity=constitution > principles > governance > SKILL=decrets ; l inferieur ne peut violer le superieur). Eprouvee, intuitive, simple, securisee. Nuance : encoder la constitution dans les skills sinon theorique.

## Détail
Kily demande comment regler le probleme d autorite des 4 fichiers (plusieurs options + reco). EXPERT-95 propose 4 options : (A) DEUX ORDRES EXPLICITES : documenter separement ordre d execution (SKILL->governance->principles->identity, le concret d abord) ET ordre d autorite (identity->principles->governance->SKILL, le fondamental prime). Honnete mais 2 ordres a retenir. (B) PYRAMIDE-FONDATION : un seul schema, identity = socle en bas qui contraint, SKILL au sommet visible ; l autorite monte de la fondation. Clair mais contre-intuitif (fondation=autorite). (C) HIERARCHIE DES NORMES / CONSTITUTIONNELLE (pyramide de Kelsen) : identity.md=Constitution (inviolable) > principles.md=droits fondamentaux/valeurs > governance.md=lois > SKILL.md=decrets/procedures. Regle universelle : une norme inferieure ne peut violer une superieure (un decret ne peut contredire la constitution). (D) Garder le schema de Kily + 1 clause de preseance minimale (en cas de conflit, identity>principles>governance>SKILL). RECOMMANDATION = C (+ visuel de B). Pourquoi : modele eprouve mondialement, intuitif (tout le monde comprend constitution > decret), resout nativement les conflits (norme contraire = nulle), UNIQUE donc simple (anti-complexite), securise (ethique prime). REGLE DE CONFLIT commune = coeur : le plus fondamental l emporte. NUANCE PRATIQUE (honnete) : sur Claude, c est SKILL.md qui est charge et a la preseance operationnelle -> la constitution (identity/principles) doit etre ENCODEE/referencee dans les skills (l organe lit identity/principles AVANT d appliquer SKILL), sinon la hierarchie reste theorique.
