Le but n est PAS l autonomie/le nombre maximal mais la performance FIABLE alignee. Implication pour NEXUS : garder peu d organes bien concus (95/97/92 = sain) plutot qu empiler. Un agent bien concu > 15 agents. Memoire : les agents a memoire persistante font +40% sur les taches entreprise.
